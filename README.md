# Dart Speedrun

### Dart Checklist for Starting Flutter Development

* **Basic Types**

  * int, double, String, bool
  * List, Map, Set
  * var, dynamic, Object

* **Null Safety**

  * nullable types (`T?`)
  * null assertion (`!`)
  * null-coalescing (`??`, `??=`)
  * `late`
  * `required`

* **Control Flow**

  * if / else
  * switch / case
  * for, for-in
  * while, do-while
  * break, continue

* **Functions**

  * arrow functions (`=>`)
  * positional parameters
  * named parameters
  * default values
  * required parameters

* **Object-Oriented Programming**

  * classes
  * constructors
  * this keyword
  * final vs const
  * extends
  * implements
  * abstract classes
  * override

* **Collections & Iterables**

  * map
  * where
  * firstWhere
  * forEach
  * reduce
  * fold

* **Asynchronous Programming**

  * Future
  * async / await
  * then
  * try / catch
  * Stream

* **Imports & Packages**

  * import / export
  * pubspec.yaml
  * dependencies / dev_dependencies

* **Mixins & Extensions**

  * mixins
  * extensions

* **Enums**

  * enum
  * enhanced enums
  * switch with enums
